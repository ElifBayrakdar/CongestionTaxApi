# CongestionTaxApi
This is an application for the calculation of tax fees for vehicles. 
### Those vehicles are supported:
* Motorbike = 0,
* Tractor = 1,
* Emergency = 2,
* Diplomat = 3,
* Foreign = 4,
* Military = 5,
* Car = 6

* Repository of tax fees will be added
* Coverage will be increased
